== Installation ==
1. 先安装WooCommerce插件
2. 安装 "xunhu-wechat-payment-for-WooCommerce" 插件并激活
3. 在WP开放平台注册账号
4. 绑定收款微信账号 
5. 创建网站应用，获取APPID和secret
6. 在插件设置中填入APPID和secret
